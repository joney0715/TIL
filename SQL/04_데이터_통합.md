# 01 함수를 사용해서 집계

## 1-1 데이터 통합

- 테이블에서 하나의 칼럼을 가져왔을 때 같은 데이터가 있는 경우, `DISTINCT`를 사용하면 데이터를 하나로 정리할 수 있음
- `DISTINCT`는 중복된 데이터를 하나로 통합
- 칼럼에 어떤 데이터가 있는지만 알고 싶은 경우 사용
- `SELECT`한 결과를 통합
- 칼럼을 여러개를 썼을 경우, 두 칼럼의 조합이 중복된 것을 정리

```sql
SELECT
	DISTINCT pref
FROM
	inquiry;
```



| 원본   | DISTINCT 사용 |
| ------ | ------------- |
| 서울시 | 서울시        |
| 충청도 | 충청도        |
| 경기도 | 경기도        |
| 서울시 |               |
| 서울시 |               |
